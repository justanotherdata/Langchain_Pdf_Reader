Steps to run this app:

In a new terminal run the steps sequentially.

1)Create a virtual environment in the root folder and activate it

2)Run the command to install dependencies of this project
    pip install -r requirements.txt

3)Once done run the command: streamlit run main.py

4)Your app has started. To close press ctrl+c


Note: You will need to make a .env file to setup your 'OPENAI_API_KEY'
